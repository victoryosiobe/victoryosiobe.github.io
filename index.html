Kingchop

Kingchop is a flexible, static text tokenizer for English, built with node JavaScript, and it is still maintained till date. I have been able to push better features despite English inconsistencies.

Kingchop as at of v1.0.6 (Aurora), had paragraph, sentence, sub-sentence, words, tokenizing abilities, with extra correct text method.
